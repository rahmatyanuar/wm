webMethods Adapter 9.6 for Apache Kafka

Release Notes

Software AG is pleased to introduce webMethods Adapter 9.6 for Apache Kafka. This adapter helps your organization to seamlessly integrate with Kafka, a distributed streaming platform. webMethods Adapter for Apache Kafka provides a graphical interface to develop services that enable you to publish and subscribe to a stream of records. In addition, the cluster aware listener service allows you to subscribe to topics and receive only one copy of the messages across the cluster.

Release Highlights

Publish and Subscribe

webMethods Adapter for Kafka provides templates to publish and subscribe messages. The adapter provides templates that are easy to configure and a simple administration user interface to modify producer and consumer configurations. The adapter also supports partitions for publish and subscribe.

IS Document Types and Custom Serializer

webMethods Adapter for Kafka provides out-of-the-box functionality to publish and receive IS documents without coding. You may also optionally plug-in your own custom serializer for publish and subscribe.

Listener

Kafka adapter listener allows you to continuously receive messages in real time and process them immediately. The listener is cluster aware and ensures that you receive the messages only once.

Security

webMethods Adapter for Kafka provides SSL support for secured connections.

For additional information, see the webMethods Adapter 9.6 for Apache Kafka Installation and Users Guide and webMethods Adapter 9.6 for Apache Kafka Readme. You can access these documents, as well as all product documentation, from the webMethods area on the Software AG Documentation website.
