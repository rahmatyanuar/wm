wm.tn.docattr:add
Adds a new document attribute.
Input Parameters
attribute
Object The document attribute to add. The document must be an instance of a subclass of com.wm.app.tn.doc.BizDocAttribute.
Output Parameters
updateCount
String The number of attributes that the service added. Values indicate:
1 - The service added the attribute.
0 - The service did not add the attribute.
