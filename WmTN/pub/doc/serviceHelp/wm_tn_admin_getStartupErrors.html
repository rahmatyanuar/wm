wm.tn.admin:getStartupErrors
Returns the error count and the list of errors that occur during Trading Networks startup.
Input Parameters
None.
Output Parameters
DBErrorCount
String Indicates the number of database errors that occurred during Trading Networks startup.
DBErrors
String (optional) List of database errors that occurred during Trading Networks startup.
ErrorCount
String Indicates the number of general errors (other than database errors) that occurred during Trading Networks startup.
Errors
String (optional) List of general errors that occurred during Trading Networks startup.
